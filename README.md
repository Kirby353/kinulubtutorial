# ムカデロボットの作成
##　注意点
説明書に沿って組み立てていく。
完成した際にギア組違いでうまく動かなかった。
胴体部分のウォームギアの接触が不良の場合回転がスムーズではない。
この場合に組み立てるとモーターが回らなくなる。
ギアの組み立ての際、にねじ止めが弱すぎると、ギアボックスが浮き動力の伝わるギアが回ってくれない。
電池の減り具合に注意すること。
胴体部のウォームギアの滑らかさは回すと明らかに違う。
